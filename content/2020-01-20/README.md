### startsWith

![startsWith](https://github.com/emersonbroga/social-media-snippets/blob/master/content/2020-01-20/1080x1080-starts-with.png)

#### Código para fácil acesso:

```js
const band = "Iron Maiden";

band.startsWith("Iron"); // true

band.startsWith("iron"); // false
```

startsWith

O método startsWith determina se uma string começa com outra string.
Fique atento, pois esse método é case-sensitive.
