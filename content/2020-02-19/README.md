### Promise.Resolve();

![Promise.Resolve()](https://github.com/emersonbroga/social-media-snippets/blob/master/content/2020-02-19/1080x1080-promise-resolve.png)

#### Código para fácil acesso:

```js
Promise.resolve('emersonbrogadev');
```

Promise.Resolve();

O Promise.Resolve(), retorna uma Promise, que se resolve com o valor informado.
Isso é util quando precisamos encadear promises usando o .then();
