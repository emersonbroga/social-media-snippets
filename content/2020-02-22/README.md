### Promise.Reject();

#### Código para fácil acesso:

```js
Promise.reject(new Error('Poxa vida!'));
```

Promise.Reject();

Assim como o Promise.Resolve(), temos também o Promise.Reject(), retorna uma Promise, que é rejeitada com o valor informado.
