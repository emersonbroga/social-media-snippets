### resto

![resto](https://github.com/emersonbroga/social-media-snippets/blob/master/content/2020-01-18/1080x1080-remainder.png)

#### Código para fácil acesso:

```js
12 % 5 // 2

-1 % 2 // -1

1 % -2 // 1

NaN % 2 // NaN

1 % 2; // 1

2 % 3 // 2

-4 % 2 // 0

5.5 % 2 // 1.5
```

Na matemática, temos o resto (remainder), ou seja o que sobra de uma divisão.
No JavaScript, podemos conseguir o resto usando o sinal de %.

Sendo assim você consegue facilmente aplicar em seu código.
